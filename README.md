# Overview

A python3 script to perform Diffie-Hellman brute-force attack.

## Usage

```
DH_brute-force.py b^emodk=r
```

* b: the base (b>0)
* e: the exponent (leave as e)
* k: the divisor (k>0)
* r: the remainder (r>0)
